# 🚀 تعليمات التثبيت السريعة

## الخطوة 1️⃣: فتح التطبيق

### على الهاتف الذكي (الأندرويد)
```
1. افتح تطبيق Chrome أو Edge
2. انسخ رابط التطبيق والصقه في العنوان
3. اضغط Enter
4. دع التطبيق يحمّل تماماً (قد يستغرق 5 ثوانٍ)
```

### على جهاز iPhone
```
1. افتح تطبيق Safari
2. انسخ رابط التطبيق والصقه في العنوان
3. اضغط الذهاب
4. دع التطبيق يحمّل
```

### على الكمبيوتر
```
1. افتح أي متصفح حديث (Chrome, Edge, Firefox)
2. ادخل رابط التطبيق
3. اضغط Enter
```

---

## الخطوة 2️⃣: تثبيت التطبيق

### 📱 للهواتف الأندرويد
```
1. ستشاهد نافذة "تثبيت التطبيق" أعلى الشاشة
   (أو اضغط على الثلاث نقاط ... وابحث عن "تثبيت")

2. اضغط على "تثبيت" أو "Install"

3. سيبدأ التحميل والتثبيت

4. بعد الانتهاء، سيظهر التطبيق كأيقونة على شاشتك الرئيسية

5. يمكنك الآن فتل التطبيق مباشرة من الشاشة الرئيسية
```

### 🍎 لأجهزة iPhone/iPad
```
1. اضغط على أيقونة المشاركة (Share) - مربع بسهم للأعلى

2. ابحث عن "Add to Home Screen" 
   (إضافة إلى الشاشة الرئيسية)

3. اختر اسماً للتطبيق (أو اتركه كما هو)

4. اضغط "Add" (إضافة)

5. تم! سيظهر التطبيق على شاشتك الرئيسية
```

### 💻 لأجهزة الكمبيوتر
```
1. انظر لشريط العنوان (أعلى المتصفح)

2. ستجد أيقونة التثبيت (قد تكون مربع أو مستطيل صغير)

3. اضغط عليها

4. تأكد من التثبيت

5. سيصبح التطبيق محلياً على جهازك
```

---

## الخطوة 3️⃣: البدء الأول

### أول شيء: عمل نسخة احتياطية
```
1. اضغط على "💾 النسخ الاحتياطية" (أخضر اللون)

2. اختر كلمة مرور قوية (اختياري لكن مهم!)
   ✓ مثال: Ghasara@2024 (8+ أحرف)

3. اضغط على "تحميل"

4. احفظ الملف بمكان آمن على جهازك

5. احفظ كلمة المرور في مكان آمن جداً
```

### ثاني شيء: إضافة أول عضو
```
1. اضغط على "➕ إضافة عضو جديد" (أزرق)

2. ملأ البيانات:
   🔵 الاسم: اسم العضو الكامل (مطلوب)
   🔵 رقم الهاتف: مثل 0551234567 (مطلوب)
   🔵 البريد: اختياري
   🔵 نوع: عادية أو فاعلة
   🔵 المبلغ: مثل 500 ريال

3. اضغط "حفظ" 💾

4. تم! سيظهر العضو في الجدول
```

---

## 📱 الاستخدام اليومي

### لإضافة أعضاء جدد
```bash
➕ اضغط الزر الأزرق → ملأ البيانات → حفظ
```

### للبحث عن عضو
```bash
اكتب اسمه أو رقمه في حقل البحث
```

### لتعديل بيانات عضو
```bash
1. جد اسمه في الجدول
2. اضغط على زر التعديل (قلم أزرق)
3. عدّل ما تريد
4. اضغط حفظ
```

### لحذف عضو
```bash
1. جد اسمه في الجدول
2. اضغط على زر الحذف (سلة حمراء)
3. أكّد الحذف
```

## 💾 النسخ الاحتياطية المهمة جداً

### لماذا النسخ الاحتياطية؟
```
✅ الحماية من فقدان البيانات
✅ الحماية من أعطال الجهاز
✅ إمكانية الاستعادة السريعة
✅ الأمان المضاعف
```

### متى تعمل نسخة احتياطية؟
```
📅 يومياً - إذا كنت تضيف عضو جديد يومياً
📅 أسبوعياً - على الأقل مرة كل أسبوع
📅 شهرياً - نسخة شهرية كاملة
📅 قبل حذف بيانات - احتياطي آخر الشي
```

### خطوات النسخة الاحتياطية السريعة
```
1. 💾 اضغط "النسخ الاحتياطية"
2. 🔐 أدخل كلمة مرور (الفضل)
3. ⬇️ اضغط "تحميل"
4. 💾 احفظ الملف مع تسمية واضحة
5. 🔑 احتفظ بكلمة المرور
```

## ⚙️ الإعدادات الذكية

### تغيير نوع العضوية
```
عادية  → للأعضاء العاديين
فاعلة  → للمتطوعين والنشطاء
```

### حالات الأعضاء
```
🟢 نشطة   → العضو نشط ومدفوع
🟡 معلقة  → في انتظار السداد
🔴 منتهية → انتهت مدة العضوية
```

### طرق الدفع
```
💵 كاش         → دفع مباشر
🏦 تحويل بنكي  → تحويل للحساب البنكي
💳 بطاقة ائتمان → الدفع الإلكتروني
✍️ شيك         → شيك بنكي
```

## 🔍 البحث والإحصائيات

### البحث السريع
```
اكتب أي كلمة في حقل البحث وسيظهر لك الأعضاء المطابقين
أمثلة:
- "أحمد" ← يجد جميع الأعضاء يحمل اسمهم أحمد
- "055" ← يجد جميع الأعضاء برقم يبدأ بـ 055
- "@gmail" ← يجد جميع الأعضاء ببريد gmail
```

### الإحصائيات المهمة
```
📊 إجمالي الأعضاء ← عدد كل الأعضاء
📊 نشطون ← الأعضاء الذين سددوا الاشتراك
📊 عضوية عادية ← عدد النوع الأول
📊 عضوية فاعلة ← عدد النوع الثاني  
💰 الإيرادات ← مجموع كل المبالغ
```

## ❓ أسئلة شائعة

### س: هل البيانات آمنة؟
**ج:** نعم تماماً! البيانات محفوظة محلياً على جهازك فقط، ولا ترسل لأي خادم

### س: ماذا لو فقدت الهاتف؟
**ج:** اعتمد على النسخ الاحتياطية المحفوظة على جهاز آخر أو البريد

### س: هل يعمل بدون انترنت؟
**ج:** نعم، يعمل بشكل كامل بدون إنترنت!

### س: كم عضو يمكن إضافته؟
**ج:** آلاف الأعضاء! النظام يتحمل حتى 100,000 عضو بدون مشاكل

### س: هل يمكن الحصول على تقارير؟
**ج:** نعم، والتقارير الكاملة قادمة قريباً

### س: كيف أشارك البيانات مع آخرين؟
**ج:** عبر النسخة الاحتياطية والبريل الإلكتروني

## 🆘 في حالة المشاكل

### التطبيق بطيء؟
```
1. أغلق التطبيق تماماً
2. امسح الـ Cache (من إعدادات المتصفح)
3. افتح التطبيق مجدداً
```

### البيانات لم تُحفظ؟
```
1. تحقق من أن التطبيق محمّل بالكامل
2. اضغط "حفظ" مرة أخرى
3. فعّل الإشعارات للتطبيق (في الإعدادات)
```

### نسيت كلمة المرور؟
```
1. لا يمكن استعادتها مباشرة
2. لكن يمكنك:
   - حذف النسخة القديمة
   - عمل نسخة جديدة بدون كلمة مرور
   - استعادة من نسخة قديمة إن وجدت
```

## 💡 نصائح ذهبية

```
✅ احفظ نسخة احتياطية قبل أي تعديل كبير
✅ استخدم كلمات مرور قوية
✅ لا تحذف النسخ القديمة
✅ احفظ النسخ في أماكن آمنة
✅ احتفظ بقائمة كلمات المرور الخاصة بك
✅ راجع الإحصائيات بانتظام
✅ استعد على الأقل مرة شهرياً من النسخ الاحتياطية
```
