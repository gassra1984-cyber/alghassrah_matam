# نظام عضوية مأتم الغسرة - نسخة أولية

## التشغيل على الكمبيوتر
1. فك ضغط الملف.
2. افتح المجلد في VS Code.
3. شغّل `index.html` باستخدام إضافة Live Server.

أو من داخل المجلد:
```bash
python -m http.server 8000
```
ثم افتح:
`http://localhost:8000`

## ملاحظات
- البيانات تحفظ داخل IndexedDB في الجهاز.
- النسخ الاحتياطي يتم من زر "تنزيل نسخة احتياطية".
- يمكن استعادة البيانات من ملف JSON.
- لتثبيت التطبيق على الهاتف يجب فتحه عبر HTTPS أو localhost.
