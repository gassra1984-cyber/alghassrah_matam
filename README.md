# 📋 نظام عضويات مأتم الغسرة - PWA

نظام متقدم لإدارة عضويات مأتم الغسرة مع إمكانيات متقدمة وآمنة للغاية.

## ✨ المميزات الرئيسية

### 🔵 العمل بدون انترنت
- **تطبيق ويب تقدمي (PWA)** - يعمل بدون إنترنت تماماً
- كل البيانات محفوظة محلياً على الجوال
- Service Worker يوفر تجربة سلسة

### 💾 نظام التخزين المتقدم
- **IndexedDB** - قاعدة بيانات قوية محلية
- تخزين آمن للبيانات الحساسة
- أداء سريع جداً حتى مع آلاف السجلات

### 🛡️ النسخ الاحتياطية المتعددة
- **تحميل/تنزيل** - احفظ النسخة بصيغة JSON على جهازك
- **إرسال بريد** - أرسل النسخة عبر البريد الإلكتروني
- **حماية بكلمة مرور** - اختياري لكن مهم جداً
- **جاهز للسحابة** - سهل التكامل مع خدمات التخزين السحابي

### 👥 إدارة العضويات
- **نوعي عضوية**: عادية وفاعلة
- **نظام الدفع**: اشتراكات سنوية
- **الحالات**: نشطة / معلقة / منتهية
- **طرق الدفع**: كاش / تحويل بنكي / بطاقة ائتمان / شيك

### 📊 إحصائيات فورية
- إجمالي الأعضاء
- الأعضاء النشطون
- توزيع الأنواع
- إجمالي الإيرادات

### 🔍 بحث وتصفية
- البحث السريع بالاسم أو الهاتف أو البريد
- تصفية فورية للنتائج

## 📱 كيفية التثبيت

### على الهاتف الأندرويد

1. **افتح التطبيق في المتصفح**
   - انسخ رابط التطبيق وافتحه في Chrome أو أي متصفح حديث

2. **ثبّت التطبيق**
   - سيظهر زر "تثبيت" أو "إضافة إلى الشاشة الرئيسية"
   - اضغط على الزر
   - اختر "تثبيت" من النافذة المنبثقة

3. **استمتع**
   - التطبيق سيصبح تطبيق محلي على جهازك
   - يعمل بدون إنترنت

### على هاتف iPhone/iPad

1. **افتح في Safari**
2. **اضغط زر المشاركة** (Share icon)
3. **اختر "أضف إلى الشاشة الرئيسية"**
4. **أعطِ التطبيق اسماً واضغط إضافة**

### على الكمبيوتر

1. **افتح المتصفح**
2. **انتقل لرابط التطبيق**
3. **انقر على أيقونة التثبيت** في شريط العنوان
4. **اختر تثبيت**

## 🚀 البدء السريع

### 1. إضافة عضو جديد

```
1. اضغط على زر "إضافة عضو جديد" 🔵
2. املأ البيانات:
   - الاسم (مطلوب)
   - رقم الهاتف (مطلوب)
   - البريد الإلكتروني (اختياري)
3. اختر نوع العضوية:
   - عادية ➜ للأعضاء العاديين
   - فاعلة ➜ للمتطوعين والنشطاء
4. أدخل المبلغ والمعلومات الأخرى
5. اضغط "حفظ" 💾
```

### 2. البحث عن عضو

- استخدم حقل البحث أعلى الجدول
- ابحث بالاسم أو الهاتف أو البريد
- النتائج تتحدث فوراً

### 3. تعديل بيانات عضو

```
1. جد العضو في الجدول
2. اضغط على زر التعديل (قلم أزرق)
3. عدّل البيانات المطلوبة
4. اضغط "حفظ"
```

### 4. حذف عضو

```
1. جد العضو في الجدول
2. اضغط على زر الحذف (صندوق أحمر)
3. أكّد الحذف
```

## 💾 النسخ الاحتياطية - شرح مفصل

### ⬇️ تحميل النسخة الاحتياطية

**الخطوات:**
1. اضغط على "💾 النسخ الاحتياطية"
2. (اختياري) أدخل كلمة مرور لحماية البيانات
3. اضغط على "تحميل"
4. سيتم حفظ ملف JSON على جهازك

**الملف المحفوظ:**
- اسمه: `backup-[timestamp].json`
- الصيغة: JSON (نص عادي)
- يحتوي على كل البيانات

**مثال للبيانات المُحفوظة:**
```json
{
  "timestamp": 1704067200000,
  "date": "2024/01/01 12:00:00",
  "data": [
    {
      "id": "M1704067200000",
      "name": "أحمد محمد",
      "phone": "0551234567",
      "email": "ahmed@example.com",
      "type": "فاعلة",
      "amount": 500,
      "status": "نشطة"
    }
  ],
  "version": "1.0"
}
```

### ⬆️ استعادة من نسخة احتياطية

**الخطوات:**
1. اضغط على "💾 النسخ الاحتياطية"
2. إذا كانت النسخة محمية، أدخل كلمة المرور
3. اضغط على "استعادة"
4. اختر الملف JSON
5. سيتم تحميل جميع البيانات

**تنبيه مهم:**
- تأكد من اختيار الملف الصحيح
- البيانات الحالية ستُستبدل بالبيانات المُستعادة
- احفظ نسخة قبل الاستعادة

### 📧 إرسال عبر البريد الإلكتروني

**الخطوات:**
1. اضغط على "💾 النسخ الاحتياطية"
2. اضغط على "بريد"
3. سيفتح تطبيق البريل الافتراضي
4. أكمل الإرسال

**ملاحظات:**
- تتضمن النسخة إحصائيات مختصرة
- تحتاج إلى تطبيق بريد مثبت
- يمكنك إضافة متلقين إضافيين

### ☁️ خيار السحابة (قريباً)

سيتم إضافة تكامل مع:
- Google Drive
- Dropbox
- OneDrive
- أي خدمة سحابية أخرى

## 🔒 حماية البيانات والأمان

### تشفير البيانات
- البيانات مشفرة محلياً باستخدام IndexedDB
- النسخ الاحتياطية يمكن حمايتها بكلمة مرور

### كيفية حماية النسخة الاحتياطية

```
1. قبل التحميل أو الإرسال
2. أدخل كلمة مرور قوية (يفضل 8+ أحرف)
3. ستُحفظ مشفرة في النسخة
4. عند الاستعادة، ستُطلب نفس الكلمة
```

### نصائح الأمان

✅ **افعل:**
- احفظ نسخ احتياطية منتظمة (أسبوعياً على الأقل)
- استخدم كلمات مرور قوية
- احفظ النسخ في أماكن آمنة
- قم بنسخ احتياطية من النسخ الاحتياطية

❌ **تجنب:**
- عدم الحفظ الدوري
- استخدام كلمات مرور ضعيفة
- حذف النسخ القديمة
- الاعتماد على نسخة واحدة فقط

### الأرقام المعروضة

## ⚙️ الإعدادات المتقدمة

### تنظيف البيانات
```
إذا أردت مسح جميع البيانات:
1. استخدم أدوات مطور المتصفح
2. اذهب للـ Storage > IndexedDB
3. احذف قاعدة البيانات MembershipDB
```

### تحديث التطبيق
- التطبيق يتحقق من التحديثات تلقائياً
- سيُخبرك بالتحديثات الجديدة
- لا تحتاج إلى أي إجراء يدوي

## 🐛 حل المشاكل الشائعة

### المشكلة: لا أستطيع تثبيت التطبيق
**الحل:**
- استخدم متصفح حديث (Chrome, Edge, Firefox, Safari)
- تأكد من رابط HTTPS
- امسح الـ Cache وحاول مجدداً

### المشكلة: البيانات لم تُحفظ
**الحل:**
- تحقق من مساحة التخزين المتاحة
- أعد تحميل الصفحة
- جرب استعادة من نسخة احتياطية

### المشكلة: أنسيت كلمة المرور
**الحل:**
- لا توجد طريقة لاستعادتها
- استخدم نسخة احتياطية بدون حماية إن أمكن

### المشكلة: التطبيق بطيء
**الحل:**
- حذّ البيانات القديمة
- امسح الـ Cache
- أعد تثبيت التطبيق
